1. **Q:** What is cloud computing?  
   **A:** Delivering computing resources like servers, storage, and software over the internet.

2. **Q:** Name the three main cloud service models.  
   **A:** IaaS, PaaS, SaaS.

3. **Q:** Difference between public and private cloud?  
   **A:** Public is shared; private is dedicated to one organization.

4. **Q:** What is TCO in cloud computing?  
   **A:** Total Cost of Ownership â€“ the full cost of cloud adoption.

5. **Q:** What is the difference between scalability and elasticity?  
   **A:** Scalability is capacity growth over time; elasticity is on-demand resource adjustment.

6. **Q:** Define cloud deployment models.  
   **A:** Public, private, hybrid, and community clouds.

7. **Q:** What is a cloud tenant?  
   **A:** An individual or organization using shared cloud resources.

8. **Q:** What is multi-tenancy in cloud?  
   **A:** Multiple tenants share infrastructure while keeping data isolated.

9. **Q:** What are cloud service benefits?  
   **A:** Cost savings, flexibility, scalability, and faster deployment.

10. **Q:** Difference between SaaS and traditional software?  
    **A:** SaaS is hosted on the cloud and accessed via browser; traditional software is installed locally.

## ðŸŸ¡ Intermediate Level (10 Q&A)
1. **Q:** How does cloud computing optimize costs?  
   **A:** Pay-as-you-go, reduced CapEx, efficient scaling.

2. **Q:** Explain vendor lock-in.  
   **A:** Dependence on a single cloud provider, making switching difficult.

3. **Q:** Key compliance and security considerations?  
   **A:** Data privacy, encryption, access control, auditing, disaster recovery.

4. **Q:** Difference between CapEx and OpEx?  
   **A:** CapEx: upfront investment; OpEx: ongoing operational costs.

5. **Q:** What is cloud governance?  
   **A:** Policies and procedures to manage cloud resources, cost, and risk.

6. **Q:** Difference between IaaS, PaaS, and SaaS with example?  
   **A:** IaaS: virtual machines; PaaS: app platforms; SaaS: cloud applications like Gmail.

7. **Q:** What is a hybrid cloud?  
   **A:** Combination of private and public clouds for flexibility.

8. **Q:** How do you measure ROI in cloud projects?  
   **A:** Compare TCO savings, efficiency gains, and business benefits vs. cost.

9. **Q:** What are cloud SLAs?  
   **A:** Service Level Agreements guaranteeing uptime, support, and performance.

10. **Q:** Explain disaster recovery in cloud.  
    **A:** Backup and recovery processes ensuring business continuity during failures.

## ðŸ”´ Advanced Level (10 Q&A)
1. **Q:** How do you assess TCO for cloud migration?  
   **A:** Include CapEx, OpEx, labor, downtime, and projected ROI.

2. **Q:** How does cloud affect business agility?  
   **A:** Enables faster deployment, scaling, and responsiveness to market changes.

3. **Q:** What are common cloud adoption risks?  
   **A:** Security breaches, compliance violations, vendor lock-in, cost overruns.

4. **Q:** How to mitigate cloud vendor lock-in?  
   **A:** Use multi-cloud strategy, open standards, and contract flexibility.

5. **Q:** What is cloud cost optimization strategy?  
   **A:** Rightsizing resources, automation, reserved instances, and monitoring usage.

6. **Q:** Explain hybrid cloud management challenges.  
   **A:** Integration, data consistency, security, and compliance across environments.

7. **Q:** How can cloud governance improve decision-making?  
   **A:** Provides visibility, control, and compliance for strategic resource allocation.

8. **Q:** Describe a scenario requiring risk assessment.  
   **A:** Hosting sensitive customer data in public cloud; evaluate security, compliance, and backup.

9. **Q:** How do you ensure cloud compliance for global operations?  
   **A:** Follow regional regulations, implement encryption, audit trails, and data localization policies.

10. **Q:** How does cloud technology impact vendor negotiations?  
    **A:** Knowledge of TCO, ROI, and service models allows better contract terms and cost leverage.
